🌰 场景：根据水果的特征判断它是“苹果”还是“橙子”
第一步：把现实对象 → 转成向量（特征向量化）
我们测量每个水果的两个特征：

重量（克）
颜色红度（0~1，0=完全不红，1=非常红）
水果	重量 (g)	红度	标签
A	150	0.9	苹果
B	160	0.85	苹果
C	130	0.2	橙子
D	140	0.15	橙子
?	155	0.88	???（待预测）
我们将每条数据表示为一个2维向量：

A → [150, 0.9]
B → [160, 0.85]
C → [130, 0.2]
D → [140, 0.15]
未知样本 → [155, 0.88]
